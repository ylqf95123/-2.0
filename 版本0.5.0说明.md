# 🐱 版本 0.5.0 - 展开节点修复版

## 🎯 核心问题

之前版本的问题：
1. ✅ 能找到"全能视频"节点
2. ✅ 能点击"全能视频"节点
3. ❌ **但无法展开"全能视频"节点**
4. ❌ 导致找不到子目录"精武门"

**根本原因**：`expandNode` 函数没有找到百度网盘的真实展开按钮！

## 🔍 真相

从你提供的截图（30.png）我发现了百度网盘的展开按钮：

```html
<em class="b-in-blk plus-icon-operate">
  <em>
```

关键类名：**`plus-icon-operate`** 和 **`em-b-in-blk`**

之前的选择器完全没有匹配到这个元素！

## 🔧 解决方案

### 1. 更新展开按钮选择器

```javascript
expander: [
  '.plus-icon-operate',              // ← 新增！百度网盘专用
  '.em-b-in-blk.plus-icon-operate',  // ← 新增！完整类名
  '[class*="plus-icon"]',            // ← 新增！模糊匹配
  '[class*="switch"]',
  '[class*="expand"]',
  '[class*="arrow"]',
  '[class*="caret"]',
  '[role="button"]',
],
```

### 2. 完全重写 `expandNode` 函数

采用**4层策略**确保展开：

```javascript
function expandNode(node) {
  // 检查是否已展开
  if (node.getAttribute("aria-expanded") === "true") {
    return; // 已展开，跳过
  }

  // 策略1: 查找 plus-icon-operate（百度网盘专用）
  let expander = node.querySelector('.plus-icon-operate');
  if (expander) {
    console.log("✓ 找到 plus-icon-operate");
    expander.click(); // 点击4次确保生效
    dispatchClick(expander);
    setTimeout(() => expander.click(), 50);
    setTimeout(() => expander.click(), 100);
    return;
  }

  // 策略2: 使用标准选择器
  expander = findFirst(node, SELECTORS.expander);
  if (expander) {
    console.log("✓ 找到标准展开按钮");
    expander.click();
    dispatchClick(expander);
    return;
  }

  // 策略3: 查找所有可能包含"icon"的元素
  const candidates = node.querySelectorAll(
    '[class*="icon"], [class*="plus"], em, i, span'
  );
  for (const candidate of candidates) {
    const classes = candidate.className || '';
    if (classes.includes('icon') || classes.includes('plus')) {
      console.log("✓ 尝试候选元素:", classes);
      candidate.click();
      dispatchClick(candidate);
      break;
    }
  }

  // 策略4: 实在不行就点击节点本身
  console.log("尝试点击节点本身");
  dispatchClick(node);
  node.click();
}
```
