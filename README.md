
# 百度网盘转存目录搜索

一款油猴脚本，给百度网盘分享页的"保存到网盘"弹窗加上文件夹搜索框，不用一层层点开也能快速定位到深处的目录。

---

## 快速开始

### 1. 安装浏览器扩展

推荐使用 [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)。

### 2. 添加脚本

- **方式 A**：双击 `baidu-pan-save-search.user.js`，在弹出的 Tampermonkey 确认页点安装
- **方式 B**：打开 Tampermonkey 管理面板，新建脚本，把脚本内容粘贴进去保存

### 3. 使用方法

1. 打开任意分享链接
2. 点"保存到网盘"左边的文件夹图标
3. 在搜索框输入目录名
4. 点结果 → 弹窗会自动跳过去
5. 检查无误后点"确定"保存

---

## 功能特性

- ✅ 在转存弹窗顶部注入搜索框
- ✅ 递归扫描你的网盘文件夹结构并缓存
- ✅ 模糊搜索 + 最近使用置顶
- ✅ 点结果自动导航到目标目录

---

## 配置项

打开脚本，顶部 `CONFIG` 对象可调整：

- `maxDepth`：索引深度（默认 2 → 索引到第 3 层）
- `concurrency`：并发请求数（默认 4）
- `indexTTL`：缓存过期时间（默认 24h）

## 历史问题与修复

### 问题 1：显示"已定位"但弹窗没真正跳转

**现象**：搜得到目录，点击后提示成功，但实际路径没变。

**原因**：只匹配到文本就认为成功，没有等待百度网盘真正切换。

**解决**：
- 每次点击后验证：弹窗路径是否出现预期内容，或下一级节点是否可见
- 增加等待和重试，直到路径真的切过去，或确定失败

---

### 问题 2：提示"未找到目录节点"

**现象**：明明目录存在，却搜不到或定位不到。

**原因**：扫描范围太大，把整个弹窗都当成了目录树，反而漏掉了真实节点。

**解决**：
- 给容器打分，优先只在真正的文件树里找
- 过滤掉明显不是目录行的元素
- 减少无效重试，让失败更快暴露

---

### 问题 3：一级目录需手动展开（已修复）

**现象**：比如 `/全能视频/精武门`，脚本能搜到但一级目录 `全能视频` 经常不会自动展开，必须手动先点一下。

**原因**：展开按钮选择器没有匹配到百度网盘真实使用的 `plus-icon-operate` 类。

**解决**：
- 专门加入了 `.plus-icon-operate` 选择器优先匹配
- 找到后会多次点击确保展开成功

## 隐私说明

- API 请求都在你的浏览器里发，和你手动用网盘完全一样
- 索引数据只存在本地 Tampermonkey 存储里
- 不会发送任何数据到外部服务器
